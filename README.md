# mosypka.github.io

Personal portfolio website of Matthias Osypka — Fullstack Developer based in Geilenkirchen, NRW.

Built with Vite + React, deployed via GitHub Pages.

## Stack

- React 18 + Vite
- Vanilla CSS (CSS Variables, no framework)
- GitHub Actions for CI/CD

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build  # output → docs/
```
